# IP Address Tracking and Geo-Location Investigation System

## Objective
This project is developed for Cyber Forensic and Investigation subject.
It tracks IP addresses and displays geographical information.

## Features
- IP address validation
- Country, Region, City detection
- ISP detection
- Latitude & Longitude
- Google Maps location view

## Technology Stack
- Python
- Flask
- HTML
- IP-API

## How to Run
1. Install Python
2. Install dependencies:
   pip install flask requests
3. Run:
   python app.py
4. Open http://127.0.0.1:5000
